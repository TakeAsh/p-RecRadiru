# p-RecRadiru
Record NHK Net Radio らじる★らじる

## Usage
```
usage: ./RecRadiru.pl <area> <channel> <duration> [<title>] [<outdir>]
area: tokyo | osaka | nagoya | sendai
channel: r1 | r2 | fm
duration: minuites
```
